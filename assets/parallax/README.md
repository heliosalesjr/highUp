# Pasta de Assets de Parallax

## Arquivos Esperados

Coloque suas texturas de parallax aqui com os seguintes nomes e dimensões:

- `layer1_sky.png` - 360×1280px (Céu, velocidade 0.1x)
- `layer2_far_trees.png` - 360×640px (Árvores distantes, velocidade 0.3x)
- `layer3_mid_trees.png` - 360×480px (Árvores médias, velocidade 0.6x)
- `layer4_close_leaves.png` - 360×240px (Primeiro plano, velocidade 0.85x)

## Importante

- Todas as texturas devem ser **tileables verticalmente** (topo conecta com a base)
- Use pixel art consistente com o resto do jogo
- Veja o arquivo `PARALLAX_GUIDE.md` na raiz do projeto para detalhes completos
